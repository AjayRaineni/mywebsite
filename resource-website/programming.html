Web development

Web development is the process of building, creating, and maintaining websites It involves web design, web publishing, web programming, and database

It is classified into the Frontend and Back end development

The front is the part that is displayed on the web browser that can see and interact with that.

The front end languages

  1. .HTML (This is used to create the structure of the website)
  2. CSS(This is used to style the website )
  3. Javascript(it is used for the functionality of the website)

Bootstrap(it has the modules which can be used directly to save time and effort)

Backend is the part that the user can not see which is used to store and arrange the data

Backend languages

  • Python
  • PHP
  • Java
  • node.js
